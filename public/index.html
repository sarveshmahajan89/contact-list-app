About AppOrbit

Enterprise software should work more like Netflix. These applications and the data they rely on, should be on-demand, automated, elastic, bulletproof, responsive, and secure. Updates should take hours, not months, with zero downtime to users. Existing applications should be far less expensive, in order to shift resources to net new innovation. And all applications should be completely portable, without change, to any modern infrastructure that offers the best combination of price, reliability, availability, and speed.
